Credit Card Fraud Detection Using Machine Learning

Overview

This project evaluates and compares multiple machine learning algorithms for detecting fraudulent credit card transactions using the European Cardholders Credit Card Fraud Detection dataset.

The objective is to identify fraudulent transactions while addressing the significant class imbalance present in real-world financial data.

Dataset

Dataset Source:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

The dataset contains anonymized transaction data from European cardholders and includes both legitimate and fraudulent transactions.
